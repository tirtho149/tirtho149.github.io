## Highlights

- Built the first unified benchmark for pathway-guided cancer therapy-response modeling (BINN, GraphPath, PATH) under identical conditions.
- Multi-task pipeline over 2,622 TCGA patients across 5 cohorts using Reactome pathway-activity features.
- Best model reached 0.92 AUROC on prostate targeted-therapy prediction at 11% class prevalence.
