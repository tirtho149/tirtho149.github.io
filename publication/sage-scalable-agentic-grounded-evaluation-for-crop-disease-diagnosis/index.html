    SAGE: Scalable Agentic Grounded Evaluation for Crop Disease Diagnosis

    June, 2026

    Abstract

    SAGE is an autonomous vision-language agent for crop-disease diagnosis, built on the largest plant-disease image–symptom dataset to date: 335 crops, 1,251 disease classes, and roughly 839K images with source-grounded, expert-validated symptom descriptions. The agent reasons in explicit diagnostic steps—anatomical context, symptom-guided candidate narrowing, reference-image comparison, and an explainable trace. Symptom grounding raised diagnostic accuracy by +16.2 points on average and generalizes to unseen crops using only reference images and symptom knowledge, with no retraining. This work is the core component of the author’s M.S. thesis (co-first author).
